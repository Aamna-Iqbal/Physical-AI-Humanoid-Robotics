# Quickstart: Module 3: The AI-Robot Brain (NVIDIA Isaac™) Course Content

This guide provides a quick overview for understanding and contributing to the content of Module 3: The AI-Robot Brain (NVIDIA Isaac™) Course.

## Module Overview

Module 3 focuses on teaching AI students about advanced perception, simulation, and navigation for humanoid robots using NVIDIA Isaac, covering Isaac Sim for photorealistic simulation, Isaac ROS for hardware-accelerated VSLAM and perception pipelines, and Nav2 for humanoid navigation.
